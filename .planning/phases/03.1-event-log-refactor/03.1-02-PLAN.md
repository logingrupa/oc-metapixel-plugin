| Boundary | Description |
|----------|-------------|
| EventLogWriter → DB | Race-fence semantics depend on `insertOrIgnore` returning affected-rows as the exact race-winner signal. Misreading the return value breaks idempotency. |
| SiteResolver → October SDK | SDK may not exist at runtime (single-site install). SDK probe MUST swallow errors and return null. |
| EventLog model → polymorphic subject | `$morphTo['subject' =&gt; []]` declaration depends on column naming convention (`subject_type` + `subject_id`). Drift from these names silently breaks the relation. |
